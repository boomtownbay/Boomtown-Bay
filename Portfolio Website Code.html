Here’s how you can host your portfolio website online:

1. Host on GitHub Pages (Free & Easy)

Step 1: Create a GitHub Repository

  • Go to GitHub and sign in or create an account.
  • Click New Repository → Name it (e.g., my-portfolio).
  • Set it to Public and click Create Repository.

Step 2: Upload Your Files

  • On your local computer, create a folder and place your HTML, CSS, and JavaScript files inside.
  • In GitHub, go to your repository → Click Add file → Upload files.
  • Select your files and click Commit changes.

Step 3: Enable GitHub Pages

  • Go to Settings → Scroll down to GitHub Pages.
  • Under Source, select main (or master) branch.
  • Click Save → Your website will be live at https://yourusername.github.io/my-portfolio/.

2. Host on Netlify (Drag & Drop)

Step 1: Create an Account

  • Go to Netlify and sign up.

Step 2: Upload Your Files

  • Click Sites → Drag and drop your project folder.
  • Netlify will assign a free domain like https://your-site-name.netlify.app.

Step 3: Customize Domain (Optional)

  • Click Domain Settings to change the site name.

3. Host on Vercel (Best for Frontend)

Step 1: Create an Account

  • Sign up at Vercel using GitHub.

Step 2: Deploy Your Website

  • Click New Project → Import your GitHub repository.
  • Click Deploy, and your website will be live in minutes.

4. Host on Firebase Hosting (For Advanced Users)

Step 1: Install Firebase CLI

  • Install Node.js, then run:
    bash
    npm install -g firebase-tools
  • Login with:
    bash
    firebase login

Step 2: Initialize Firebase

  • In your project folder, run:
    bash
    firebase init
  • Select Hosting → Choose default settings.

Step 3: Deploy Your Site

  • Run:
    bash
    firebase deploy
  • Firebase will provide a live URL for your site.

Which One Should You Choose?

  • For beginners → GitHub Pages or Netlify (simple and free).
  • For custom domains → Vercel or Firebase.
  • For backend needs → Firebase Hosting.

  • Link your JavaScript file before </body> in index.html:
    html
